Principais formatações

Negrito / destaque

Nessa frase temos um termo em negrito usando a tag <b>(não semântica).

Nesta frase, tenho um termo em destaque usando a tag <strong>(semântica)

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i>(não semântica)

Nessa frase temos um termo em ênfase, usando a tag <em>(semântica)

Texto Marcado

Podemos também criar um texto marcado usando a tag <mark>

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, utilizando a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, utilizando a tag <ins>.

Existe também o sublinhado com a tag <u>(não semântica)

Texto sobrescrito

Para inserir coisas do tipo x2+3, utilizamos a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O, utilizamos a tag <sub>
